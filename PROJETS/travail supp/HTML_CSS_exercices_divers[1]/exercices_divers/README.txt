L'idée est de reproduire la zone visible de la capture d'écran. Si vous voulez tenter de reproduire la totalité du site, certaines parties seront parfois plus difficile à réaliser comme les navigations par exemple.

Dans le dossier image_overlay_demo.zip, vous pouvez trouver une démo qui met en oeuvre deux propriétés : display:none, le position en absolute et le sélecteur :hover.
Si vous le souhaitez vous pouvez tenter de reproduire la démo par vous même (mais c'est assez difficile) ou simplement étudier le code fourni.
